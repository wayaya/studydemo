FTP登陆后目录结构如下:
根目录
|- web 网站根目录,网页文件请上传到这个目录,此目录不能删除，否则网站无法访问！
|- log 是用于存放您网站每天的访问日志，请勿删除，默认日志功能不开启。
|- db 是用于在线打包/解压功能，请勿删除。
注意,您的网页必需存放在web目录下才能被访问,默认首页文件按优先顺序为index.asp,index.html,index.htm。
本网页文件名为 index.htm 是您的主机自动开设后系统生成的页面,您可以删除掉或用自己的页面将此页面覆盖。
推荐您使用CUTEFTP,Flash Fxp或者LEAPFTP等软件进行上传,
如果在上传中碰到无法列出目录等情况,请尝试着把上传软件的PASV模式关掉。
欢迎您使用我司虚拟主机。CUTEFTP软件下载，Flash Fxp 软件下载，LEAPFTP软件下载

FTP软件如何设置 ：
FTP软件有很多种，但每种软件都要设置以下几项内容：
1〕 FTP 主 机：您可以写上您的域名，如我司赠送的三级域名，或您站点主机的IP地址。
2〕 FTP用户名：您开通时填写的。
3〕 FTP 密 码：即您开通时填写的。
4〕 FTP 端 口：默认21即可。
5〕 点连接后，您将看到3个文件夹，您把网页内容上传到web文件夹里。
6〕 然后您就可以浏览您的精彩网站了！
